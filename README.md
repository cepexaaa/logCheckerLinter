## Линтер для проверки лог-записей
*** logCheckerLinter - это статический анализатор (линтер) для языка Go, который проверяет лог-вызовы на соответствие установленным правилам. Линтер поддерживает две популярные библиотеки логирования: log/slog, go.uber.org/zap ***

##### Функциональность:
- Проверка регистра первого символа
- Проверка языка сообщений (только английский)
- Обнаружение спецсимволов и эмодзи
- Выявление чувствительных данных (пароли, токены, ключи)
- Автоисправление для правила о регистре
- Гибкая настройка через конфигурационный файл


Установка:
```bash
git clone https://github.com/cepexaaa/logCheckerLinter
cd logCheckerLinter

# Сборка
go build -o loglint ./cmd/loglint

# Сборка плагина для golangci-lint
go build -buildmode=plugin -o loglint.so ./plugin
```

Можно запускать как отдельный анализатор:
```bash
go vet -vettool=$(which loglint) ./...
```

### Интеграция с golangci-lint

- Соберите плагин: `go build -buildmode=plugin -o loglint.so ./plugin`
- Запустите: `golangci-lint run`


#### Спорные вопросы и важные моменты, которые были решены в реализации:
- Полагается, что допустимы только символы ASCII - как раз английские и ещё доп символы
- Проверка на специальные символы происходит в происходит внутри проверки на английский
- Добавил проверку на то, что в логах нету email и номера карты, потому что тоже конфиденциальная информация (аналогично можено встроить проверки на другие объекты)
